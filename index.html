How To Play

Tutorial how to play in your device

  • ANDROID

    1. Download and install the latest version of Growtopia from Google Play.
    2. Open Growtopia and log in to get your items updated.
    3. Download the hosts file below.
    4. Open Virtual Hosts and tap on Select Host File and select the file you just downloaded.
    5. Turn the switch on and log in to Growtopia.

    Download host Download Virtual Host Apk

  • PC

    1. Open notepad as administrator
    2. Click on File and select Open
    3. Go to path C:\Windows\System32\drivers\etc and open Hosts file
    4. Add the server ip in thats file, Like this:
    Server Ip Address growtopia1.com
    Server Ip Address growtopia2.com
    5. Click File and select Save
    6. Now log in to Growtopia
